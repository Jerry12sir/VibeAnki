# JLPT Vocabulary Desk

Testing

Browser app for studying the two included Anki vocabulary sessions:

- Shinkanzen N3
- Kaishi 1.5k

## Features

- Separate Shinkanzen and Kaishi sessions.
- Shinkanzen can be browsed lesson by lesson.
- Quick Quiz can combine sessions and selected lessons.
- Kanji, hiragana, meaning, and example display.
- Learned state is saved in your browser with `localStorage`.
